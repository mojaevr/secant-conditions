# SP-Broyden

Магистерская диссертация: **«Исследование глобальных квазиньютоновских методов
оптимизации для задач математического программирования и вариационных
неравенств»**.

Автор: Можаев Р. М., МФТИ.
Научный руководитель: Камзолов Д. И.

## Что в работе

Диссертация строит семейство квазиньютоновских обновлений
**SP-Broyden**, сохраняющих несколько прошлых секущих условий,
и применяет его к трём классам задач:

- нелинейные системы уравнений (улучшенная ограниченная деградация
  с квадратичной зависимостью от $\|x-x^*\|$);
- безусловная оптимизация в симметричном случае (семейство SS-U
  и его частный вариант SS-SR1 с расширенным бассейном сходимости);
- сильно монотонные вариационные неравенства (алгоритм SP-AFD
  с гарантированной оптимальной скоростью $O(T^{-3/2})$).

Основная публикация, в которую входит схема VIJI-Restarted:
Agafonov A., Ostroukhov P., Mozhaev R. et al.
*Exploring Jacobian inexactness in second-order methods for variational
inequalities: lower bounds, quasi-Newton updates, and restarts.*
NeurIPS 2024. arXiv:2405.15990.

## Сборка PDF

Требуется TeX Live 2023+ с русскоязычной поддержкой (пакет `mathtext`,
`babel-russian`, T2A-энкодинг).

```bash
cd mipt_thesis_master
./build.sh       # два прогона pdflatex
```

Итог — `mipt_thesis_master/main.pdf`.

## Воспроизведение экспериментов

Ноутбуки требуют Python 3.10+ и NumPy/SciPy/matplotlib.
